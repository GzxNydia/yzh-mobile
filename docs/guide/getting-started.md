## 前序准备

你的本地环境需要安装 [yarn](https://yarnpkg.com)、[node](http://nodejs.org/) 和 [git](https://git-scm.com/)。

## 安装

```
// yarn
yarn add yzx-mobile

// npm
npm install yzx-mobile
```

## 使用

```
import React from 'react';
import { Button } from 'yzx-mobile';

class Example extends React.Component {
  render() {
    return (
      <div>
        <Button />
      </div>
    );
  }
}
```
